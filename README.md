# 🛒 Amazon Clone Website

## 📌 Project Overview

The Amazon Clone is a responsive frontend e-commerce website inspired by Amazon’s homepage. This project focuses on building a clean, structured, and interactive user interface using core web technologies.

The main goal of this project was to strengthen frontend development skills and understand how real-world e-commerce websites are designed and structured. The project includes responsive layouts, product sections, navigation components, and interactive UI elements.

---

# 🛠️ Technologies Used

- HTML5  
- CSS3  
- Flexbox  
- CSS Grid  
- JavaScript  
- DOM Manipulation  

---

# ✨ Features

- ✔️ Responsive Navigation Bar  
- ✔️ Search Bar UI

- ✔️ Product Cards with Images, Prices & Ratings  
- ✔️ Mobile Responsive Design
