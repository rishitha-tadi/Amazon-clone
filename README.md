📌 Project Overview
The Amazon Clone is a responsive frontend e-commerce website inspired by Amazon’s homepage.
This project focuses on building a clean, structured, and interactive UI using core web technologies.
The goal of this project was to strengthen my frontend development skills and understand how real-world e-commerce websites are designed and structured.

🛠️ Technologies Used
HTML5
CSS3 (Flexbox & Grid)
JavaScript (DOM Manipulation)

✨ Features
✔️ Responsive Navigation Bar
✔️ Search Bar UI
✔️ Product Listing Section
✔️ Product Cards with Images, Price & Ratings
✔️ Mobile Responsive Design

🎯 What I Learned
Creating structured layouts using Semantic HTML
Designing complex UI using Flexbox and CSS Grid
Implementing JavaScript DOM manipulation
Improving UI/UX and responsive design skills
Organizing project folders professionally

🔮 Future Improvements
🔐 User Authentication (Login/Signup)
🛍️ Cart Data Storage using localStorage
💳 Payment Gateway Simulation
🌐 Backend Integration
📦 Product Filtering & Sorting
